Веб-версия игры 2048 на чистом HTML, CSS и JavaScript.

## Реализованные функции

- Сетка 4x4, генерируемая динамически через JavaScript
- Движение и слияние плиток во всех 4 направлениях
- Появление 1-2 новых плиток (2 или 4) после каждого хода
- Корректный подсчёт и отображение очков
- Проверка окончания игры (нет возможных ходов)
- Кнопка "Начать заново" без перезагрузки страницы
- Отмена последнего хода (undo)
- Таблица лидеров: top-10, хранение в localStorage
- Ввод имени и сохранение результата при окончании игры
- Сохранение состояния игры в localStorage
- Управление стрелками на десктопе
- Виртуальные кнопки и свайпы на мобильных устройствах
- Плавная анимация движения и слияния плиток
- Уникальные цвета для каждого значения плитки
- Адаптивный дизайн (десктоп, планшет, смартфон)

## Технологии

- HTML5 (без innerHTML/outerHTML)
- CSS3 (анимации, flexbox, медиа-запросы)
- Vanilla JavaScript (ES5, без фреймворков)

## Запуск

Открыть `index.html` в браузере или запустить локальный сервер:

```bash
npx serve .
```
